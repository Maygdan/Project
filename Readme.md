# Проект «Doodle Knight: Greenpath Edition»

## 1. Описание проекта
Данный проект представляет собой кроссовер легендарной механики **Doodle Jump** и визуальной эстетики культовой игры **Hollow Knight** (локация «Зеленая тропа»). 
Игрок берет на себя роль маленького рыцаря, который должен подняться как можно выше по мшистым платформам заросшего королевства Халлоунест.

**Ключевая особенность:** Игра сочетает в себе  геймплей аркады и графику, реализованную на фреймворке **WPF**.

---

## 2. Архитектура решения
Проект строго разделен на два программных модуля согласно принципам SOLID и требованиям ТЗ:

1.  **DoodleJump.Model (Class Library / DLL)** — содержит всю бизнес-логику игры, физические расчеты, систему координат и механизмы сохранения данных. Файлы внутри разделены на папки **Core** (сущности и логика) и **Data** (сериализация).
2.  **DoodleJump.Desktop (WPF App)** — отвечает за визуализацию, обработку пользовательского ввода и анимацию. Использует ресурсы (Assets) для создания атмосферы Greenpath.

---

## 3. Соответствие критериям оценивания (на 10 баллов)

Проект полностью реализует все требования Продвинутой, Дополнительной части:

*   **Интерфейсы (2):** `IUpdatable` (объекты с логикой обновления) и `IPlatform` (свойства игровых площадок).
*   **Абстрактные классы (3):** 
    *   `GameObject` — базовый класс для всех объектов мира.
    *   `Platform` — основа для различных типов площадок.
    *   `BaseSerializer<T>` — обобщенная база для систем сохранения.
*   **Приведение типов:** В методе `Draw()` реализовано более 5 явных приведений к базовым классам и интерфейсам для обработки игровых объектов.
*   **Перегрузка методов:** В классе `Player` реализован метод `Jump()` в двух вариациях: стандартный прыжок и прыжок с заданной силой.
*   **Переопределение (Override):** Методы `Update()` и `OnCollision()` переопределены во всех наследниках (`NormalPlatform`, `TrapPlatform`, `SpringPlatform`).
*   **Перегрузка операторов:** В структуре `Vector2D` перегружены операторы `+` и `-` для удобной работы с координатами.
*   **Делегаты и События:** Использован встроенный делегат `Action` для события `OnGameOver`, связывающего Модель с Интерфейсом.
*   **Обобщенные типы (Generics):** Класс `BaseSerializer<T>` позволяет работать с любыми типами данных при сохранении.
*   **Инструменты визуализации:** Использовано более 5 различных контролов: `Canvas` (игровое поле), `DataGrid` (рекорды), `Slider` (сложность), `ComboBox` (форматы), `Viewbox` (масштабирование), `Image`.

## 4. Функциональные возможности (ТЗ)

### Базовая и дополнительная части:
*   **Физика:** Реализовано ускорение свободного падения и плавное замедление прыжка.
*   **Закольцованность:** При пересечении края экрана персонаж появляется с противоположной стороны.
*   **Partial классы:** Класс `GameEngine` разделен на части для удобства поддержки и разделения логики.
*   **Сложность:** Выбор уровня сложности (1-5) через слайдер в меню динамически меняет шанс появления платформ-ловушек.
*   **Выбор папки:** Пользователь может указать произвольную папку для хранения файлов игры.

### Продвинутая часть (Сериализация):
*   **Два формата:** Поддержка **JSON** (Newtonsoft.Json) и **XML**.
*   **Авто-конвертация:** При смене формата в главном меню программа автоматически копирует данные из текущего формата в новый.
*   **Продолжить игру:** Кнопка «Продолжить» активируется только при наличии валидного файла сохранения.
*   **Разнообразие объектов:** Добавлены `SpringPlatform` (супер-прыжок) и `TrapPlatform` (платформа-ловушка, которая мгновенно исчезает при касании).

## 5. Инструкция по использованию

1.  **Запуск:** После запуска откроется стартовое меню в стиле Hollow Knight.
2.  **Настройка:**
    *   Выберите уровень сложности (чем выше, тем больше ловушек).
    *   Выберите предпочтительный формат сохранения (JSON/XML).
    *   Укажите путь к папке сохранений (кнопка `...`).
3.  **Игра:** Нажмите «НОВАЯ ИГРА».
    *   Управление: **A / D** или **Стрелки** влево/вправо.
    *   Цель: Прыгать по мшистым платформам как можно выше. Избегайте падения за нижний край экрана.
4.  **Сохранение:** Просто закройте окно игры. Ваш текущий прогресс (позиция и очки) будет сохранен автоматически.
5.  **Рекорды:** Таблица в главном меню отображает Топ-10 ваших лучших попыток.
